e.g., rgba(255, 0, 153, 0.8) instead of rgb(255 0 153 / 80%) and hsla(150, 30%, 60%, 0.8) instead of hsl(150 30% 60% / 80%)

Note: If the color string is a link, clicking on it will convert the color instead of opening the link. To open the link, hold CTRL while clicking.
